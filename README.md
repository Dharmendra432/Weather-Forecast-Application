# Weather Forecast Application

# Github Link
https://github.com/Dharmendra432/Weather-Forecast-Application.git

## Overview
A responsive Weather Forecast Application built using JavaScript, HTML, and Tailwind CSS. It fetches real-time weather data from OpenWeatherMap API and displays current and extended forecasts.

## Features
- Search weather by city
- Use current location
- Recent cities dropdown
- Temperature unit toggle
- Extreme weather alerts
- 5-day forecast display
- Responsive UI (Mobile, Tablet, Desktop)

## Technologies Used
- HTML5
- Tailwind CSS (CDN)
- JavaScript (ES6)
- OpenWeatherMap API
